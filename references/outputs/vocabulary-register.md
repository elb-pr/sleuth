## Approved Replacement Patterns

When the default phrasing impulse arises, use these approved alternatives instead:

| Default Impulse | Approved Alternative |
|-----------------|---------------------|
| Generic significance framing | State the specific significance directly |
| Noting or pointing out | State the fact directly |
| Abstract realm/landscape framing | Use concrete location: "ground", "territory", "world" (once per script max) |
| Complexity navigation | Use specific approved verbs: "examine", "trace", "follow" |
| Woven tapestry framing | Use once per script maximum; prefer "stories", "threads", "lineage" |
| Underlining importance | "demonstrates", "reveals", "exposes" |
| Deep exploration framing | "examine", "trace", or specific action verb |
| Multi-dimensional descriptors | Use specific approved adjectives from the register above |
| Subtlety descriptors | Use specific concrete description |
| Emotional resonance framing | Use specific emotional verb: "cuts", "haunts", "burdens" |
| Generic quality praise | Remove entirely or name the specific quality |
| Turning-point framing | Use once per script maximum; vary with "decisive", "catalytic", "the moment when" |
| Unravelling metaphor | "disintegrates", "fractures", "splinters" |
| Illumination metaphor | State the revelation directly |
| Crucial role framing | State the function directly |
| "In terms of" construction | Restructure the sentence |
| Summarising cliches | Remove entirely |
