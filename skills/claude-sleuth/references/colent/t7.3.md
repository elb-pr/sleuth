# t7.3 — Universal Metadata Tagging

**Tool:** CSDb MCP

## Procedure

1. You are REQUIRED to attach the following mandatory metadata to every entity record and every relationship link:

   **Source Attribution (field-level, not record-level):**
   - Source Identifier from t5.3
   - Source Admiralty Grade (e.g., B2, F1) from t3.1b
   - If multiple sources contribute to the same field, YOU MUST record all contributing Source Identifiers and the corroboration status.

   **Temporal Metadata:**
   - `dateObserved`: when the information was current in the real world.
   - `dateCollected`: when the investigator captured it.
   - `dateRange` for relationships: start and end dates where known. Open-ended relationships carry a start date with no end date.
   - Records older than the defined currency threshold MUST be flagged as potentially stale and are REQUIRED to undergo re-verification before use in final analytical products.

   **Analyst Identity:**
   - `createUser`: analyst who created the record.
   - `lastModifiedBy`: most recent modifier, with timestamp.
   - Full review history: every analyst who has reviewed, verified, or amended the record.

   **Confidence Scoring:**
   - Entity-level confidence: High (deterministic match), Medium (probabilistic, analyst-confirmed), Low (limited corroboration).
   - Field-level confidence MUST be recorded where specific fields are less certain than the record overall.

2. It is CRITICAL that during any record merge, metadata from BOTH source records is preserved and combined. Source attributions MUST be merged, not overwritten. Conflicting field values MUST be retained with their respective source attributions — silent resolution of conflicts is FORBIDDEN.

3. ALWAYS maintain complete data lineage from any merged entity back to its original constituent source records. This lineage is REQUIRED for legal defensibility — an entity with untraceble provenance is a complete systematic failure.

4. In UK policing contexts, YOU MUST record the 5×5×5 handling code (t3.1b) as record-level metadata and enforce compartmentalisation for protected source material.
